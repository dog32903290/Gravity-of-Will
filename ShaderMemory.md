# Shader Memory Core - Project "0406: The Unburial Core"

## 核心思想 (Core Thought)

- **生命意志的對抗：** 模擬阿嬤在醫院病床上的意識場域。探討「表象（肉體與外界衝擊）」與「意志（核心自我）」的拉鋸。
- **哲學基底：** 叔本華的「生命意志 (Will to Live)」。意志是不可摧毀的奇點，而死亡與疾病是試圖掩埋光的黑色泥沼。

## 視覺與動態軌律 (Visual & Dynamic Rules)

1. **黑色泥沼 (The Mud):**
   - 黑色泥沼才是整體視覺主題。
   - 細節極其豐富，質地像是「碎裂的土地」，一塊一塊的往中間擠壓過去，沒有氣體感。

   - **向心重力：** 視覺流動方向為往中心點移動，將周圍的碎塊向內吞噬。
   - **速度連動：** 整體速度流動變得更慢。當泥沼收縮加快時，代表外部壓力增強。

2. **埋不掉的光 (The Unburial Core):**
   - 位於中心的意志奇點。光源更小兩倍，大小變化範圍很小，脈動緩慢。
   - **暈影收束：** 中間的光點暈影極少，維持高銳度的小圓球狀態。
   - **大幅度漫遊：** 意志光點會在較大的範圍內緩慢漂移漫遊。
   - **純粹的紅光：** 紅色僅存在於中間的光球，不影響黑色的泥沼。光球代表意志的奇點。
   - **放射狀粒子系統：** 明確的純白光點由中心向外放射，並具有隨機長短不一的拖尾軌跡，非均勻霧氣。
   - **物理抗衡 (Push & Friction)：** 粒子射出時會「推開」泥沼（泥沼在粒子後方產生向外退開的物理錯覺）。同時，粒子若行經泥沼板塊上方，會受到「摩擦阻力」而快速消耗動能消失，只有在裂縫間能存活較久。每一個粒子也具有獨立初始動能。
   - **應激反應 (銳利)：** 當泥沼速度加快，光點變得極其銳利（邊界收窄），試圖向外穿刺，但視覺上仍敵不過泥沼向內的速度（悲劇性抗爭）。
   - **消融擴散 (模糊)：** 當泥沼速度減慢，光點變得模糊、溫暖且向外稍微擴散（意志的片刻安息）。
   - **約束：** 所有變化必須極其微妙、緩慢，符合生理呼吸頻率，嚴禁過度誇張的特效。

3. **外界訊號 (The Signal):**
   - 醫生或家人的呼喚、外界的干擾。
   - 呈現為「網狀線條」或「漣漪」，瞬間震盪泥沼的結構，喚醒意志的定向。

## 技術架構 (Technical Architecture)

- **渲染引擎：** Three.js (WebGL).
- **核心技術：**
  - **Ping-pong FBO (雙緩衝反饋)：** 用於實現「意識的延遲感」與「泥沼的累積質感」。
  - **FBM (Fractal Brownian Motion) Noise:** 產生底層黏稠的流場。
  - **Custom Shaders:** 處理光與泥沼的非線性混合邏輯（Exp 衰減、透射感）。

## 已確認的設計決策 (Confirmed Decisions)
- **銳度連動：** 非線性（指數）映射 — `sharpness = baseSharp * exp(k * mudSpeed)`。壓力臨界點後銳度急劇攀升。
- **泥沼質感：** FBM + Domain Warping，產生岩漿般的摺疊扭曲。
- **第一版範圍：** 僅泥沼與光，不含外界訊號（網狀線條/漣漪）。

- **2026-04-06:** 視覺大更新 — 將黑色泥沼確立為視覺主體，增加碎裂土地細節（Voronoi `cellular`）。修正流動方向，讓土地一塊一塊**往中心點緩慢移動**擠壓。光源縮小並縮減閃動幅度，確立紅色僅屬於中心光點。

  - 引入 Bingham 塑性流體概念 — Voronoi 板塊差速運動（靠近中心壓得更快）
  - 每個板塊獨立漂移速度（cellHash → 0.7x~1.3x 差速）

  - 紅光暈影大幅收縮：`uBaseSharp` 28.0 + `smoothstep(0.12, 0.03)` 硬遮罩截斷擴散
  - 裂縫更銳利：`smoothstep(0.0, 0.04)` (from 0.08)
- **2026-04-06:** 光點粒子向外擠壓系統：
  - 反向採樣層 — 從靠近中心的位置讀取前一幀，內容向外傳播
  - 亮度閾值過濾（brightPass）— 只攜帶光亮內容，不帶泥沼
  - 距離衰減 `exp(-dist * 6.0)` — 粒子隨距離消散
  - Noise 散射 + 稀疏遮罩 — 有機分佈，非完美徑向
  - 泥沼向內、光粒子向外，在共享 feedback buffer 中自然交界融合
- **2026-04-06:** 交界擠壓質感（Viscous Fingering）：

  - 雙尺度 FBM 指狀滲透 — 粗手指 + 細觸鬚，扭曲距離場邊界
  - meniscus 彎月面高光 — 表面張力隆起效果，暗紅色微光
  - 泥沼在交界處被壓縮 — 裂縫變窄 (crackWidth / compression)，板塊微微提亮
- **2026-04-06:** Dolly In 透視視差：
  - 距離加權 zoom — 邊緣像素移動速度 = 中心的 4 倍（`1.0 + radialDist * 3.0`）

  - 輸出 Pass 增強隧道暗角 — 周邊額外壓暗 40%，聚焦視覺向前推進
- **2026-04-06:** 黑白化 + 光球漫遊 + 尺寸亮度耦合：
  - 全畫面轉為純黑白 — 移除所有紅色（lightColor, meniscus, particles）
  - 光球位置改由 fbm noise 驅動漫遊（±0.075 偏移，速度 0.03）
  - 所有系統自動跟隨 center 變數：泥沼流向、粒子發射、交界區、dolly
  - 尺寸-亮度耦合：breathe noise → sizeScale(0.8~1.2) + brightScale(0.4~1.0)
- **2026-04-06:** 細節精修與粒子動態升級：
  - 光球漫遊範圍擴大（`orbWander` multiplier 從 0.15 提至 0.4）。
  - 中心光點暈影減少一倍（`sharpnessAdj` 加倍，`lightMask` 界限縮緊）。
  - 對 feedback decay 引入空間與時間的 noise 擾動，使粒子向外飛射時的殘雪/拖尾具有長短不一的隨機感。

- **2026-04-06:** 粒子與泥沼的物理互動 (Viscous interaction)：
  - **粒子推遲泥沼**：將粒子的計算層往前挪，並利用 `particlePush` 使 `sampleUV` 向外擠壓，造成「粒子經過時將黑泥向兩側推開」的錯覺。
  - **極座標落後物理 (Deceleration)**：全面捨棄直角坐標的粒子生成，改採專為泥沼設計的「Polar Procedural System」。當粒子碰到黑泥（`localFriction` 大），會因為阻尼係數瞬間放大，使得粒子的**實體半徑發展（`expectedRadius`）嚴重縮水**。如此一來達成了「碰到泥塊時，粒子不但變暗，甚至物理上會直接變慢、滯留」的真實物理錯覺。
  - **速度與射程差距放大**：整體的粒子發射時間調慢了一倍 (`uTime * 0.1`)。大幅擴大 `kineticEnergy` 的極端範圍，使得微弱的粒子死在源頭，強壯的粒子能強行撕裂整片泥沼飛出界外。
  - **細膩控制**：將鍵盤左右鍵的控制力度從 `0.01` 極限下調至 `0.002`，能像調音頻一樣極度緩慢且精確地拉升粒子量。
